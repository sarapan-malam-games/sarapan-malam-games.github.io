Membuat Interaksi Rumput Bergoyang | Godot 4

September 16, 2022 ¬∑ 3 min ¬∑ 486 words ¬∑ Sarapan
Table of Contents

Agar terlihat menarik dan hidup pada sebuah game tentunya kita perlu membuat interaksi karakter ke objek lingkungan sekitar, salah satunya rumput atau dedaunan. Ditutorial kali ini kita akan membuatnya dengan cara sederhana tetapi dengan hasil yang bagus.

üñê Persiapan#

Unduh Project Template Berikut untuk memulai:

Project Template

Pada Project Template diatas sudah kita kita atur beberapa hal, seperti:

  • Collision Layer Dan Collision Mask sudah kita atur berdasarkan jenis objeknya

  • Node Player, Tilemap, Grass sudah kita siapkan, teman-teman tinggal memodifikasinya jika perlu

Struktur Project Template:

‚úçÔ∏è Mulai#

  1. Buka Project Template, kemudian buka Shader Editor (berada dipanel bawah).

    1. Klik File -> New Shader.

    2. Setelah popup Create Shader muncul, arahkan Path ke res://shaders/ dan simpan dengan nama WindSway.gdshader atau bebas sesuai keingann teman-teman.

    3. Sekarang kita perlu membuat salinan shader dari link 2D Wind Sway, open link itu kemudian copy paste kode shadernya ke file shader yang sudah kita buat sebelumnya.

    4. Kita perlu mengedit shader tesebut, dengan menambahkan variable dan statement yang kita butuhkan untuk membuat texture menjadi terlihat miring, nanti kita gunakan untuk membuat rumput bergoyang saat player menyentuhnya.

      1. Kita buat variabel baru dengan nama skew. 
        uniform float skew: hint_range(-500.0, 500.0) = 0;
      2. Kemudian kita tambahkan statement berikut pada fungsi vertex 
        if (skew != 0.0) {
  VERTEX.x += max(0.0, (1.0-UV.y) - heightOffset) * (skew / 12.0);
}
      3. Kita simpan file shadernya dengan cara Klik File -> Save File

  2. Sekarang saatnya menerapkan shader yang sudah kita buat dan membuat script untuk node Grass.

    • Menerapkan shader:

      1. Open node Grass lalu Klik node Sprite2d.
      2. Tambahkan ShaderMaterial pada node Sprite2d
      3. Klik Resource kemudian centang properti Local To Scene pada ShaderMaterial.

        ShaderMaterial termasuk type data Resource karena itu semua data didalamnya akan dibagikan kesemua instance secara default, jadi untuk membuatnya unik disetiap instance kita perlu mencetang properti Local To Scene

      4. Pada properti ShaderMaterial tambahkan shader yang sudah kita buat sebelumnya, dengan cara drag drop file shader ke properti Shader. Kita sudah mempunyai shader yang dibutuhkan untuk membuat texture menjadi miring, kalian bisa bereksperimen dengan mengubah-ubah nilai parameter pada shader tersebut. Cobalah untuk mengubah parameter Skew, menjadi 300 atau -300, nanti akan terlihat bentuk texture menjadi agak condong ke kanan atau kiri.

    • Menerapkan Script:

      1. Klik icon script pada node Grass, kemudian pada fungsi _ready tambahkan potongan kode berikut: 
        %Sprite2d.material.set_shader_parameter("offset", randi() % 3)
connect("body_entered", _on_sway)
connect("body_exited", _on_sway)
      2. Kemudian buatlah fungsi _on_sway untuk menerima signal body_entered dan body_exited 
        if body is Player:
  var direction = global_position.direction_to(body.global_position)
  var skew = clamp(remap(body.velocity.length() * sign(-direction.x), -body.SPEED, body.SPEED, min_skew, max_skew), min_skew, max_skew)

  var tween = create_tween()
  tween.tween_property(%Sprite2d.material, "skew", skew, 0.2).set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_CUBIC)
  tween.tween_property(%Sprite2d.material, "skew", 0.0, 2).set_ease(Tween.EASE_OUT).set_trans(Tween.TRANS_ELASTIC)
        Jadi saat node Player menyentuh atau keluar dari area collision node Grass, node Grass akan bereaksi dan bergerak.

  3. Sekarang buka node Level, kemudian taruh node Grass dengan drag drop ke dalamnya, taruh posisi sesuai keinginan teman-teman.

Semoga tutorial kita kali ini dapat teman-teman pahami dan dipraktekan dengan baik. Terimakasih telah membaca dan jangan lupa untuk mempraktekkannya.
